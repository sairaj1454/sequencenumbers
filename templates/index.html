Excel Sequence Number Updater

How to use:

  1. Upload your Excel file (.xlsx or .xls)
  2. The system will process the file and add sequence numbers (100, 105, 110, etc.) to rows where 'Authored' is 'Yes'
  3. Download the processed file with updated sequence numbers

Note: Your Excel file must have 'Top Family WERS Code', 'Authored', and 'Sequence Number' columns.
